Introduction

This Privacy Policy explains how the Amazon Price Tracker Bot ("the Bot") handles information. By using the Bot, you agree to this policy.

Data We Collect

The Bot does NOT collect personal data (names, emails, addresses, etc.). We store:

  • Amazon Product Links: URLs provided by users for price tracking.
  • Price Ranges: Minimum and maximum prices set by users for alerts.

How We Use Data

  • To monitor price changes on Amazon products.
  • To send price alerts via Telegram.

Third-Party Services

The Bot uses:

  • Amazon Affiliate Links: We earn commissions through Amazon Associates. Amazon’s privacy policy applies to their services.
  • GitHub Pages: Our informational website is hosted on GitHub Pages, which does not collect user data.

Data Security

All data is stored securely and is only accessible to the Bot’s administrators.
